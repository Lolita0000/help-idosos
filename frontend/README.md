# Help Idosos - Frontend

Interface web do projeto **Help Idosos**, desenvolvida com React e TypeScript, com arquitetura baseada em MVC para organização do código.

## Tecnologias

- React
- TypeScript
- Vite
- Axios

## Estrutura do Projeto

src/
 ├─ components/
 ├─ controllers/
 ├─ models/
 ├─ services/
 ├─ views/
 └─ main.tsx

### Organização MVC (adaptada para React)

- **Models** → Tipagens e interfaces
- **Views** → Telas da aplicação
- **Controllers** → Regras entre View e Services
- **Services** → Comunicação com a API
